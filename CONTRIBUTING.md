# Contributing to spacy-service

This project came about due to personal requirements. Consequently, bug fixes are very welcome, but feature requests are unlikely to be acted upon. You are of course welcome to fork your own version.


## Workflow

- Fork the repository
- Create a branch (`git checkout -b bugfix/nasty-bug`)
- Make your changes
- Commit your changes (`git commit -m 'Fixed that nasty bug'`)
- Push to the branch (`git push origin bugfix/nasty-bug`)
- Open a Pull Request
